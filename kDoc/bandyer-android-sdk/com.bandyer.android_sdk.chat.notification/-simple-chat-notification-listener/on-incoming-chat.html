onIncomingChat

fun onIncomingChat(chat: IncomingChat, isDnd: Boolean, isScreenLocked: Boolean): Unit

This method will be called when an incoming chat has been received. Here it is possible to launch the incoming chat as activity, notification or not display it at all.

Parameters

chat - Incoming chat

isDnd - True if the device is in do not disturb mode, false otherwise

isScreenLocked - True if the device has the screen locked,false otherwise
