onIncomingFile

abstract fun onIncomingFile(file: IncomingFile, isDnd: Boolean, isScreenLocked: Boolean): Unit

This method will be called when an incoming file has been received. Here it is possible to display or not the notification.

Parameters

file - Incoming file

isDnd - True if the device is in do not disturb mode, false otherwise

isScreenLocked - True if the device has the screen locked,false otherwise
