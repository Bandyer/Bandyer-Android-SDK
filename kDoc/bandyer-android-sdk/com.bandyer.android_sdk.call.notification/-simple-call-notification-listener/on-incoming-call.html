onIncomingCall

fun onIncomingCall(call: IncomingCall, isDnd: Boolean, isScreenLocked: Boolean): Unit

This method will be called when an incoming call has been received. Here it is possible to launch the incoming call as activity, notification or not display it at all.

Parameters

call - Incoming call

isDnd - True if the device is in do not disturb mode, false otherwise

isScreenLocked - True if the device has the screen locked,false otherwise
