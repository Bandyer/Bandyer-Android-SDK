onUploadingFile

abstract fun onUploadingFile(uploadingFile: UploadingFile, isDnd: Boolean, isScreenLocked: Boolean): Unit

This method will be called when a file is being uploaded. Here it is possible to display a notification. It is highly recommended to do display it.

Parameters

isDnd - True if the device is in do not disturb mode, false otherwise

isScreenLocked - True if the device has the screen locked,false otherwise
