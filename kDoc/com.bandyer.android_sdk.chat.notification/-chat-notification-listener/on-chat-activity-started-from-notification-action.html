onChatActivityStartedFromNotificationAction

abstract fun onChatActivityStartedFromNotificationAction(chatInfo: ChatInfo, chatIntentOptions: ChatIntentOptions): Unit

This method will be called before the Chat Activity is going to be displayed. This may occur when the Incoming Chat has been launched as activity or after an interaction with the notification displayed.

Parameters

chatInfo - Information about the chat

chatIntentOptions - Chat options to be used when launching the activity
