withAudioUpgradableCallCapability

fun withAudioUpgradableCallCapability(callCapabilities: CallCapabilities, callOptions: CallOptions): ChatIntentOptions

Audio call capability means that you can initialize an audio/video call from chat activity. Upgradable means that the call will be initialized as audio call but can be upgraded to video if selected camera button from call action items.

Parameters

callOptions - call options for the call from the chat UI such as recording,...

callCapabilities - call capabilities such as whiteboard,file sharing, ...

Return
ChatIntentOptions
