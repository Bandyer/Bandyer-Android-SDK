onCallActivityStartedFromNotificationAction

abstract fun onCallActivityStartedFromNotificationAction(callInfo: CallInfo, callIntentOptions: CallIntentOptions): Unit

This method will be called before the Call Activity is going to be displayed. This may occur when the Incoming Call has been launched as activity or after an interaction with the notification displayed.

Parameters

callInfo - Information about the call

callIntentOptions - Call options to be used when launching the activity
