keepListeningforIncomingCallsInBackground

fun keepListeningforIncomingCallsInBackground(enable: Boolean): Builder

This feature allows Bandyer SDK to keep listening for incoming calls even in background mode. Please Keep in mind that incoming calls during background state should always be routed from push notification to prevent unexpected behaviours and keep battery consumption under control.

Parameters

enable - enable background listening for notifications.

Return
Builder
