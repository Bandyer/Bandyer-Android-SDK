keepListeningForEventsInBackground

fun keepListeningForEventsInBackground(enable: Boolean): Builder

This feature allows Bandyer SDK to keep listening for events in background mode. Please Keep in mind that listening for events during background will lead to unexpected behaviours and drains of battery life. To handle events in background you should handle notifications correctly via push services.

Parameters

enable - enable background listening for events.

Return
Builder
