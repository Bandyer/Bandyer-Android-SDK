DeviceInfo

object DeviceInfo

Helper class for accessing hardware specifications, including the number of CPU cores, CPU clock speed and total available RAM.

Properties

DEVICEINFO_UNKNOWN

 val DEVICEINFO_UNKNOWN: Int

The default return value of any method in this class when an error occurs or when processing fails (Currently set to -1). Use this to check if the information about the device in question was successfully obtained.

cpuMaxFreqKHz

 val cpuMaxFreqKHz: Int

Method for reading the clock speed of a CPU core on the device. Will read from either /sys/devices/system/cpu/cpu0/cpufreq/cpuinfo_max_freq or /proc/cpuinfo.

numberOfCPUCores

 val numberOfCPUCores: Int

Reads the number of CPU cores from the first available information from /sys/devices/system/cpu/possible, /sys/devices/system/cpu/present, then /sys/devices/system/cpu/.
