cpuMaxFreqKHz

val cpuMaxFreqKHz: Int

Method for reading the clock speed of a CPU core on the device. Will read from either /sys/devices/system/cpu/cpu0/cpufreq/cpuinfo_max_freq or /proc/cpuinfo.

Return
Clock speed of a core on the device, or -1 in the event of an error.
